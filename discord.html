Bot Commands

More commands coming soon!

Fantasy League

/fantasy leaderboard

View the top 10 fantasy teams

/fantasy join

Get the link to join the league

/fantasy link-league [join-code]

Link a fantasy league (Admin)

/fantasy setup-channel [channel]

Set channel for updates (Admin)

/fantasy set-time [time] [timezone]

Set weekly post time (Admin)

/fantasy unlink

Disconnect the fantasy league (Admin)

Pro League

/standings [league]

View current season standings (MLV/LOVB)

/team roster [league] [team]

View a team's roster

Games

/games upcoming [league]

Next 5 upcoming games (MLV/LOVB)

/games results [league]

Last 5 completed games with scores

/games live [league]

Currently live games

Statistics

/stat leaders [category] [league] [stat-type] [calc-type] [year]

View top 10 stat leaders

Categories: Player, Team

Stats: Aces, Kills, Digs, Assists, Blocks, Efficiency

Calc Types: Total (default), Per Set

Transactions (Admin Only)

/transactions enable [channel] [league]

Enable transaction updates

/transactions disable [channel] [league]

Disable transaction updates

Automatically posts new player transactions every minute

Settings (Admin Only)

/settings allow-channel [channel]

Enable bot in a channel

/settings block-channel [channel]

Disable bot in a channel

/settings list-channels

View all enabled channels
